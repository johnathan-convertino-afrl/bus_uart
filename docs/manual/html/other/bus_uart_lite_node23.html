Module Documentation

up_uart_lite is the module that integrates the AXIS UART core. This includes FIFO's that have there inputs/outputs for data tied to registers mapped in the uP bus. The uP bus is the microprocessor bus based on Analog Devices design. It resembles a APB bus in design, and is the bridge to other buses BUS UART LITE can use. This makes changing for AXI Lite, to Wishbone to whatever quick and painless.

axi_lite_uart_lite module adds a AXI Lite to uP (microprocessor) bus converter. The converter is from Analog Devices.

wishbone_standard_uart_lite module adds a Wishbone Standard to uP (microprocessor) bus converter. This converter was designed for Wishbone Standard only, NOT pipelined.





The next sections document these modules. up_uart_lite contains the register map explained, and what the various bits do.
